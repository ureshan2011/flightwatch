FlightWatch

Daily fares for the Christchurch ↔ Colombo corridor, with a forecast-driven buy / wait signal.
Updated automatically once a day via GitHub Actions. Fares are scraped from Google Flights across multiple browser fingerprints. Signals come from a quantile gradient-boosting model trained on each route's own price history, with a heuristic fallback while history is thin — informational only, verify the live fare before booking. Data is open: see the data/ folder.
