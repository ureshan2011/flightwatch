Daily fares for the Christchurch ↔ Colombo corridor — airlines, stops and flight times for every offer, with a forecast-driven buy / wait signal.
Updated automatically once a day via GitHub Actions. Fares are scraped from Google Flights across multiple browser fingerprints — every offer (airline, stops, duration, price) is stored as open CSV in data/. Signals come from a quantile gradient-boosting model trained on each route's own price history, with a heuristic fallback while history is thin.
